Romanian Linked Open Data Map

What it does?
  • Displays on a Google Map all resources from data.gov.ro that have physical address
  • Displays resources that are nearby to the user
  • Displays information about each resource, taken from other online sources, for example each city population, taken from geonames.org
What open datasets uses?
  • Romanian Hospitals
  • Romanian Pharmacies
  • Romanian Museums
  • More to come...Stay tuned.
